Anime Discord Bot



Invite Bot

Mission Statement

As to appeal to anime enthusiasts, the objective of this project is to create a Discord bot that provides anime-based utilities such as tracking and recommending show lists and displaying top current shows. The need for this project is essential to Discord users who hope to keep up with their shows, expand their show list by recommendations, and see what new anime everyone else is watching. This project is beneficial to fans as it will propagate the information they want to see and the data provided will allow for the user to figure out what anime they want to see or keep up with. If this project does not get built, that will be a missed opportunity for creating a Discord community unified by a common interest such as anime.

Commands


Help

A command that serves as a directory. All commands and details to run them are shown here. It will also show who requested the command.

Create List

A command that creates an ID for the user in the database, enabling the user to input their anime list.

Save List

A command that allows for the user to input their list of anime.

Show List

A command that shows the list of anime the user has inputted.

Delete Anime

A command that allows the user to delete animes the user no longer wants in their list.

Delete List

A command that allows for the user to delete the user's entire anime list.

Recommend

A command that suggests the user other animes related to animes the user is interested in.

Clear List

A command that allows the bot to delete the user's user ID from the database and remove the user's entire anime list.

Poll

A command that enables the user to put their own questions and other users will be able to answer them.

Relevant Work

A command that displays all related work to an anime.

Anime Meme

A command that displays a random meme from Reddit.

Anime Search

A command that shows the user the anime they have searched and will prompt an overview of the anime.

Manga Search

A command that shows the user the manga they have searched and will prompt an overview of the manga.

Set Episode

A command that sets the episodes the user is currently watching.

All Ranking

A command that shows the user the ranking based on myanimelist score.

Most Popular

A command that shows the user the ranking based on myanimelist views and following.

Music

For this command, there are multiple commands that the user may use to their own liking. The commands are Add, Play, Stop, Resume, Queue, Remove, View, and Skip.
• The Add command enables the user to add a song they prefer by adding a link of that song.
• The Play command enables the user to play a song that they had added to their queue.
• The Stop command enables the user to pause a song that they were listening to.
• The Resume command enables the user to play a song again that they had paused.
• The View command enables the user to view the song currently playing.
• The Skip command enables the user to skip a song. However, for this command a poll will be presented and users will be asked to vote on whether they would like to skip or continue listening to the song playing.
