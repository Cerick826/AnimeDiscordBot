Anime Discord Bot

Contributors: Erick Contreras, Vong Chen, Nurhaliza Hassan, Corey New, and William Ye

Project Description - As to appeal to anime enthusiasts, the objective of this project is to create a Discord bot that provides anime-based utilities such as tracking and recommending show lists and displaying top current shows.

Commands

Help

Details: Opens the help menu

Usage: !help <command>

Create List

Alias: `createlist`, `Createlist`, `CreateList`

Details: Creates a list in the database

Usage: !createList

Save List

Alias: `savelist`, `Savelist`, `SaveList`

Details: Saves anime to list

Usage: !saveList <anime name>

Show List

Alias: `showlist`, `Showlist`, `ShowList`

Details: Shows your list

Usage: !showList

Delete Anime

Alias: `delanime`, `Delanime`, `DelAnime`

Details: Deletes an anime from list

Usage: !delAnime <anime name>

Delete List

Alias: `deletelist`, `Deletelist`, `DeleteList`

Details: Deletes list from database

Usage: !deleteList

Recommend

Alias: `delanime`, `Delanime`, `DelAnime`

Details: Deletes an anime from list

Usage: !delAnime <anime name>

Clear List

Alias: `delanime`, `Delanime`, `DelAnime`

Details: Deletes an anime from list

Usage: !delAnime <anime name>
